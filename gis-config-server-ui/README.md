## 3. 静态资源存储访问
前端开发时，静态资源存储需遵循以下规则：

```
1. 静态资源文件放在src/main/resources/static目录下；
2. 页面文件放在src/main/resources/resources目录下；
```

## 4. 视图模版引擎static为
一般把页面放在templates中，静态资源放在static中。页面引用这些静态资源时,注意相对路径，因为//*，所以不需要引用到static那级。
